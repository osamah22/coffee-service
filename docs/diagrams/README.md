# Diagrams

This directory contains Excalidraw diagram sources and SVG exports used by the Markdown docs.

- Edit `generate-excalidiagrams.mjs` when changing the generated diagram set.
- Keep the matching `*.svg` files committed so Markdown renderers can display the diagrams.
- Run `node diagrams/generate-excalidiagrams.mjs` from `docs/` to regenerate the current sources and exports.
